MSIM 580 Assignment Three: Crossword Puzzle Designer

The solution to the Crossword Puzzle Designer problem consists of two java files located in
<this folder>/crosswordDesign
File 1: Crossword.java
File 2: Word.java

These files can be compiled using the following command line command from <this folder>/CrosswordDesign
javac Crossword.java Word.java
The Crossword file contains the main method and can be executed with the following command from <this folder>
java crosswordDesign.Crossword

In order to compile Java Development Kit (JDK) is required
The latest Standard Edition JDK can be downloaded from Oracle
https://www.oracle.com/java/technologies/javase-jdk15-downloads.html
The path to the JAVA bin file may also need to be set using the following command line command from <this folder>/crosswordDesign
set path=C:\Program Files\Java\<jdk version>\bin

The user will be prompted for the following information: 
Select a puzzle file
Select a dictionary file
Add any custom words
Select search heuristics

Alternatively the folder containing this README file can be imported into an IDE and the program can be executed through the IDE
This program was written using the Eclipse IDE

The folder containing this file aslo contains the input files used for this assignement and a results folder containing the output files resulting from generating solutions to the crossword puzzles
